# m-julian.github.io
Github Pages of m-julian

_includes: boilerplate code that includes <head> and navigationbar
_layouts: Layouts that jekyll uses depending on the type of page published
assets: images and other things that pages will use
collections: contains _notebooks which has all the jupyter notebooks (which have been converted with the jupyter_notebook_convert.py script)
contents: contains the contents page (a page that shows all different notebook pages)
css: contains css for website
favicon: containts favicon of different sizes
texts: raw notebooks and other files that are then converted to html to be displayed
_config.yml: jekyll configuration file
